HOW WE WORK
The 3 things we check on every deal.

Before a package ends up in Boarding Pass or Deal Alert, we run it through three filters. If it fails any one, it doesn't make the list.

1. Live pricing.

We don't trust PR numbers. We check the price on the actual booking site on the day we publish — and take a screenshot. If the price has moved, the writeup says so.

2. Real inclusions.

"All-inclusive" means different things at different resorts. We read the fine print: what counts as drinks (is premium liquor extra?), do transfers cost extra, are resort fees included, how much is tipping. Then we list what you actually get.

3. Historical range.

A $499 Cancun trip sounds great until you learn that resort runs $489 half the year. We track prices over time so we can tell you whether a "deal" is actually a drop — or just a normal-low.

If those three check out, we write it up. If they don't, we either watch-list it or skip it.
